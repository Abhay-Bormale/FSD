Basic HTML Tags and Attributes

Tag / Attribute Use Example
html Root element of an HTML page html ... /html
head Contains metadata and title head title Page /title /head
body Contains visible content body Hello World /body
h1 to h6 Headings from large to small h1 Heading /h1
p Paragraph p This is a paragraph /p
br Line break Hello br World
a Creates a hyperlink a href="page.html" Link /a
img Displays an image img src="image.jpg" alt="Image"
href Specifies link destination href="page.html"
src Specifies image source src="photo.png"
alt Alternate text for image alt="Profile Image"
table Creates a table table ... /table
tr Table row tr ... /tr
td Table data cell td Data /td
th Table header cell th Name /th
